## 二、核心模块详细说明

### 2.1 远程外呼（16 人天）

**功能：** 接收后台下发被叫号码，自动跳转拨号界面并模拟拨号

**技术实现：**
- 部署常驻后台 Agent 服务（Service）
- 集成无障碍服务（AccessibilityService）实现模拟点击
- MQTT/WebSocket 长连接接收指令
- 适配厂商：保活、自启动、后台锁定、电池白名单
- 指令回执机制（成功/失败/超时）
- 失败重试 + 异常兜底

**核心风险：**
- 后台服务被系统查杀
- 不同安卓版本及厂商 ROM 兼容性差异
- 长连接网络不稳定

---

### 2.2 通话录音（17 人天）

**功能：** 监听电话状态，自动录音并上传后台

**技术实现：**
- 申请 `RECORD_AUDIO`、`READ_PHONE_STATE`、`WRITE_EXTERNAL_STORAGE` 权限
- 监听 `TelephonyManager.CALL_STATE_IDLE/OFFHOOK/RINGING`
- 本地加密存储（AES）
- 挂断后自动上传后台（断点续传）
- 关联：通话记录、坐席ID、线索ID
- 支持在线回放、下载

**核心风险：**
- 部分厂商（华为/小米）限制通话录音
- 录音文件大，上传并发压力大
- 录音丢失或上传失败

---

### 2.3 短信收发监听（19 人天）

**功能：** 监听短信数据库/广播，抓取收发内容同步后台

**技术实现：**
- 监听 `CONTENT://sms/` 数据库变化（ContentObserver）
- 监听 `android.provider.Telephony.SMS_RECEIVED` 广播
- 抓取：收发内容、号码、时间戳
- 实时同步后台（MQTT/WebSocket）
- PC端支持查询、检索、导出

**核心风险：**
- 部分手机拦截短信监听
- 数据库监听不及时
- 同步延迟

---

### 2.4 电子围栏 — 离场锁机（23 人天）

**功能：** GPS 定时上报，超出设定半径自动锁机

**技术实现：**
- 后台配置中心坐标（经纬度）+ 半径（<500米）
- 定时获取 GPS/基站定位（Android LocationManager）
- GPS漂移平滑滤波（卡尔曼滤波）
- 定位数据存入时序数据库
- 后台计算距离判断是否超出围栏
- 超出则触发锁机指令
- 防误锁策略（连续3次超标才触发）

**核心风险：**
- GPS漂移导致误锁机
- 定位延迟导致围栏判断失效
- 锁机后无法紧急解锁

---

### 2.5 APP黑白名单控制（19 人天）

**功能：** 限制手机可运行的应用

**技术实现：**
- 监听 `android.intent.action.PACKAGE_ADDED/REMOVED`
- 监听 `android.intent.action.MAIN` 启动事件
- 白名单模式：仅允许名单内应用运行
- 黑名单模式：禁止运行、自动闪退
- 设备管理器（DevicePolicyManager）锁定任务
- 防包名伪装绕过

**核心风险：**
- 系统级 APP（如Settings）无法拦截
- 包名伪装规避检测
- 监听不及时导致应用已启动

---

### 2.6 截屏限制（18 人天）

**功能：** 禁止截屏，截屏时显示黑屏

**技术实现：**
- 设置 `WindowManager.LayoutParams.FLAG_SECURE`
- 全局拦截截屏操作
- 触发截屏时显示黑屏并上报后台
- 第三方截屏工具防绕过（后期迭代）

**核心风险：**
- 部分厂商ROM不支持FLAG_SECURE
- 第三方截屏工具可绕过
- 悬浮窗权限被禁用

---

### 2.7 全局水印（15 人天）

**功能：** 悬浮窗展示水印，截屏带水印

**技术实现：**
- 使用 `WindowManager` 添加悬浮窗
- 水印内容：员工号 + 设备号 + 时间（每分钟更新）
- 覆盖所有界面（Activity/-dialog/状态栏）
- 截屏时水印一同截取
- 权限申请（SYSTEM_ALERT_WINDOW）

**核心风险：**
- 水印被应用遮挡
- 部分界面无法覆盖
- 悬浮窗权限被用户禁用

## 三、保活方案（关键）

手机端所有功能依赖后台Agent常驻运行，保活是核心技术挑战：

### 3.1 保活手段

| 手段 | 说明 |
|-----|------|
| 前台Service | 设置 `FOREGROUND_SERVICE` + 通知栏常驻 |
| 无障碍服务 | 监听用户操作，维持进程优先级 |
| 电池白名单 | 引导用户加入电池白名单 |
| 自启动 | 监听 `BOOT_COMPLETED` 开机自启 |
| 厂商定制 | 华为/小米/OPPO后台锁定配置 |

### 3.2 适配工作量

| 厂商 | 适配重点 |
|-----|---------|
| 华为 | 青oning一键清理/后台管理深度定制 |
| 小米 | 神隐模式/自启动管理/电量优化 |
| OPPO | 冻结后台/纯净模式 |
| vivo | 后台管理/低电量模式 |

## 四、接口清单（手机端）

| 接口方向 | 内容 |
|--------|------|
| **手机→后台** | 心跳、GPS定位、SIM状态、录音文件、短信记录、通话记录 |
| **后台→手机** | 外呼指令、锁机指令、策略更新、围栏配置 |
| **长连接** | MQTT Topic 订阅/发布，指令下发与回执 |
