# MDM 手机管控平台

> 企业统一管理员工手机，支持远程外呼、短信管控、安全策略下发与执行

## 项目结构

```
MDM手机管控平台/
├── README.md
└── 远程控制手机端功能需求与实现方案.md
```

## 背景

本项目为企业级 MDM（Mobile Device Management）解决方案，聚焦安卓设备统一管控能力建设。

## 核心功能模块

| 模块 | 说明 |
|-----|------|
| 拨打电话 | 远程外呼、脱敏展示、通话录音、线索弹屏 |
| 发送短信 | PC远程收发、敏感词拦截、频次管控、重复监控 |
| 安全管控 | SIM卡监控、APP黑白名单、截屏限制、电子围栏、全局水印、轨迹查询 |
| 工程支撑 | 需求设计、联调测试、厂商适配、验收部署 |
