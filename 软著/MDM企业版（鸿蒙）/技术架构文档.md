## 1. 项目概述

### 1.1 项目背景
为企业提供基于鸿蒙系统的设备远程管控解决方案，通过 MQTT 长连接实现云端指令下发与状态上报。

### 1.2 核心功能
- ✅ 设备远程锁定/解锁、恢复出厂设置、重启、重置密码  
- ✅ 应用安装/卸载/黑名单管理  
- ✅ 网络管控（WiFi、移动数据、蓝牙、GPS）  
- ✅ 硬件外设限制（相机、麦克风、USB、截图等）  
- ✅ 设备状态监控（电量、网络、在线状态）  
- ✅ MQTT 长连接 + 智能心跳 + 自动重连  
- ✅ 电源优化管理（动态调整心跳与重连策略）

### 1.3 技术栈

| 层级 | 鸿蒙实现 |
|------|----------|
| **语言** | ArkTS (TypeScript 超集) |
| **MQTT** | `@ohos/mqtt` (v5.0.0+) |
| **MDM 能力** | `@kit.MDMKit` (EnterpriseAdminExtensionAbility) |
| **HTTP** | `@ohos.net.http` |
| **本地存储** | `@ohos.data.preferences` / `@ohos.data.relationalStore` |
| **后台服务** | `@ohos.ability.backgroundTaskManager` |
| **UI** | ArkUI (声明式) |

## 3. MQTT 通信层

### 3.1 配置参数 (`MqttConfig.ets`)

```typescript
export class MqttConfig {
  static readonly SERVER_URI = 'tcp://192.168.1.199:1883'
  static readonly CLIENT_ID_PREFIX = 'mdm_siyu_'

  static readonly CONNECTION_TIMEOUT = 30      // 秒
  static readonly KEEP_ALIVE_INTERVAL = 60     // 秒
  static readonly CLEAN_SESSION = true
  static readonly AUTO_RECONNECT = true

  static readonly USERNAME = 'admin'
  static readonly PASSWORD = 'admin'

  // 遗嘱消息
  static readonly WILL_TOPIC = 'mdm/report'
  static readonly WILL_QOS = 1
  static readonly WILL_RETAINED = true

  static readonly PUBLISH_QOS = 1
  static readonly SUBSCRIBE_QOS = 1
}
```

### 3.2 MQTT 客户端封装 (`MqttClient.ets`)

使用 `@ohos/mqtt` 库（需在 `oh-package.json5` 中添加 `"@ohos/mqtt": "5.0.0"`）：

```typescript
import mqtt from '@ohos/mqtt'

export class MqttClientManager {
  private client: mqtt.MqttClient | null = null
  private deviceId: string = ''
  private isConnected: boolean = false
  private listeners: ((connected: boolean) => void)[] = []

  initialize(deviceId: string) {
    this.deviceId = deviceId
    this.client = mqtt.createMqttClient(MqttConfig.SERVER_URI, {
      clientId: MqttConfig.CLIENT_ID_PREFIX + deviceId,
      cleanSession: MqttConfig.CLEAN_SESSION,
      keepAlive: MqttConfig.KEEP_ALIVE_INTERVAL
    })
    this.client.on('connect', this.onConnect.bind(this))
    this.client.on('message', this.onMessage.bind(this))
    this.client.on('close', this.onClose.bind(this))
    this.client.on('error', this.onError.bind(this))
  }

  connect() {
    if (!this.client) return
    this.client.connect({
      username: MqttConfig.USERNAME,
      password: MqttConfig.PASSWORD,
      connectTimeout: MqttConfig.CONNECTION_TIMEOUT * 1000,
      reconnectPeriod: 5000,
      will: {
        topic: MqttConfig.WILL_TOPIC,
        payload: JSON.stringify({ action: 'offline', code: this.deviceId }),
        qos: MqttConfig.WILL_QOS,
        retained: MqttConfig.WILL_RETAINED
      }
    })
  }

  subscribe(topic: string, qos: number = MqttConfig.SUBSCRIBE_QOS) {
    this.client?.subscribe(topic, qos)
  }

  publish(topic: string, message: string, qos: number = MqttConfig.PUBLISH_QOS): boolean {
    if (!this.isConnected) return false
    this.client?.publish(topic, message, qos)
    return true
  }

  disconnect() { this.client?.disconnect() }

  private onConnect() {
    this.isConnected = true
    this.listeners.forEach(fn => fn(true))
    // 订阅设备专属主题
    this.subscribe(`mdm/${this.deviceId}`)
    this.subscribe(`device/config/${this.deviceId}`)
  }

  private onMessage(topic: string, payload: string) {
    // 触发外部回调
  }

  private onClose() {
    this.isConnected = false
    this.listeners.forEach(fn => fn(false))
  }

  private onError(err: any) { console.error('MQTT error:', err) }

  addConnectionListener(listener: (connected: boolean) => void) {
    this.listeners.push(listener)
  }
}
```

## 4. 消息协议

### 4.1 主题规划

| 主题 | 方向 | 说明 |
|------|------|------|
| `mdm/{deviceId}` | 服务器 → 设备 | 接收指令 |
| `device/config/{deviceId}` | 服务器 → 设备 | 接收配置更新 |
| `mdm/report` | 设备 → 服务器 | 遗嘱（离线通知） |
| `mdm/command/{deviceId}` | 设备 → 服务器 | 命令执行结果 |
| `mdm/{deviceId}/heartbeat` | 设备 → 服务器 | 心跳数据 |
| `mdm/{deviceId}/alert` | 设备 → 服务器 | 告警信息 |

### 4.2 命令消息格式（服务器 → 设备）

```json
{
  "commandId": "uuid-123456",
  "action": "lock",
  "params": {
    "mainText": "设备已锁定",
    "subText": "请联系管理员"
  },
  "timestamp": 1704067200000
}
```

### 4.3 支持的命令列表

| action | 说明 | 参数 |
|--------|------|------|
| `lock` | 锁定屏幕 | mainText, subText |
| `unlock` | 解锁 | - |
| `wipe` | 恢复出厂 | - |
| `reboot` | 重启 | - |
| `reset_password` | 重置密码 | password |
| `install_app` | 安装应用 | apkPath / bundleName |
| `uninstall_app` | 卸载应用 | bundleName |
| `disable_wifi` / `enable_wifi` | 禁用/启用 WiFi | - |
| `disable_mobile_data` / `enable_mobile_data` | 禁用/启用移动数据 | - |
| `set_gps` | GPS 开关 | enabled (boolean) |
| `set_bluetooth` | 蓝牙开关 | enabled |
| `disable_camera` / `enable_camera` | 相机禁用/启用 | - |
| `get_device_info` | 获取设备信息 | - |
| `get_app_list` | 获取应用列表 | - |
| `sync_config` | 同步配置 | config 对象 |

## 5. 命令处理系统

### 5.1 命令解析器 (`CommandParser.ets`)

```typescript
export class CommandMessage {
  commandId: string
  action: string
  params: Record<string, any> | null
  timestamp: number

  static fromJson(json: string): CommandMessage | null {
    try {
      const obj = JSON.parse(json)
      if (!obj.commandId || !obj.action) return null
      return new CommandMessage(
        obj.commandId,
        obj.action,
        obj.params || null,
        obj.timestamp || Date.now()
      )
    } catch { return null }
  }

  getString(key: string, def = ''): string { return this.params?.[key] ?? def }
  getBoolean(key: string, def = false): boolean { return this.params?.[key] ?? def }
  getNumber(key: string, def = 0): number { return this.params?.[key] ?? def }
}
```

### 5.2 命令结果封装 (`CommandResult.ets`)

```typescript
export class CommandResult {
  commandId: string
  action: string
  success: boolean
  message: string
  data: Record<string, any> | null
  timestamp: number

  static success(cmdId: string, action: string, msg = '执行成功'): CommandResult {
    return new CommandResult(cmdId, action, true, msg)
  }
  static failure(cmdId: string, action: string, msg: string): CommandResult {
    return new CommandResult(cmdId, action, false, msg)
  }
  toJson(): string { return JSON.stringify(this) }
}
```

### 5.3 命令处理器 (`CommandHandler.ets`)

```typescript
import { DeviceController } from '../mdm/DeviceController'
import { AppManager } from '../mdm/AppManager'
import { RestrictionManager } from '../mdm/RestrictionManager'
import { InfoQuerier } from '../mdm/InfoQuerier'

export class CommandHandler {
  private deviceCtrl = new DeviceController()
  private appMgr = new AppManager()
  private restrictMgr = new RestrictionManager()
  private info = new InfoQuerier()

  handle(cmd: CommandMessage): CommandResult {
    switch (cmd.action) {
      case 'lock': return this.deviceCtrl.lock(cmd)
      case 'unlock': return this.deviceCtrl.unlock(cmd)
      case 'wipe': return this.deviceCtrl.wipe(cmd)
      case 'reboot': return this.deviceCtrl.reboot(cmd)
      case 'reset_password': return this.deviceCtrl.resetPassword(cmd)
      case 'install_app': return this.appMgr.install(cmd)
      case 'uninstall_app': return this.appMgr.uninstall(cmd)
      case 'disable_wifi': return this.restrictMgr.setWifiDisabled(cmd, true)
      case 'enable_wifi': return this.restrictMgr.setWifiDisabled(cmd, false)
      case 'disable_camera': return this.restrictMgr.setCameraDisabled(cmd, true)
      case 'enable_camera': return this.restrictMgr.setCameraDisabled(cmd, false)
      case 'get_device_info': return this.info.getDeviceInfo(cmd)
      case 'get_app_list': return this.info.getAppList(cmd)
      default: return CommandResult.failure(cmd.commandId, cmd.action, `未知命令: ${cmd.action}`)
    }
  }
}
```

## 6. 心跳与保活机制

### 6.1 智能心跳调整 (`HeartbeatAdjuster.ets`)

```typescript
import battery from '@ohos.battery'
import power from '@ohos.power'

export class HeartbeatAdjuster {
  getIntervalMs(): number {
    const batteryInfo = battery.getBatteryInfo()
    const level = batteryInfo.batterySOC
    const isCharging = batteryInfo.chargingStatus === 1
    const isScreenOn = power.isScreenOn()

    if (isCharging && level > 50) return 60000      // 1分钟
    if (level < 20 && !isCharging) return 300000    // 5分钟
    if (!isScreenOn && !isCharging) return 600000   // 10分钟
    return 120000 // 2分钟
  }

  buildHeartbeatMsg(deviceId: string): string {
    const batteryInfo = battery.getBatteryInfo()
    return JSON.stringify({
      deviceId,
      batteryLevel: batteryInfo.batterySOC,
      isCharging: batteryInfo.chargingStatus === 1,
      isScreenOn: power.isScreenOn(),
      networkType: this.getNetworkType(),
      timestamp: Date.now()
    })
  }

  private getNetworkType(): string {
    // 实际使用 @ohos.net.connection 获取
    return 'WiFi'
  }
}
```

### 6.2 心跳定时器启动

```typescript
startHeartbeat(deviceId: string, publish: (topic: string, msg: string) => void) {
  const adjuster = new HeartbeatAdjuster()
  const send = () => {
    const msg = adjuster.buildHeartbeatMsg(deviceId)
    publish(`mdm/${deviceId}/heartbeat`, msg)
  }
  send()
  setInterval(() => {
    const interval = adjuster.getIntervalMs()
    send()
    // 动态调整下次间隔（简单实现中可重新设置setInterval）
  }, adjuster.getIntervalMs())
}
```

## 7. 重连策略

### 7.1 智能重连延迟 (`ReconnectManager.ets`)

```typescript
export class ReconnectManager {
  private disconnectTime = 0

  getDelayMs(): number {
    if (this.disconnectTime === 0) this.disconnectTime = Date.now()
    const minutes = (Date.now() - this.disconnectTime) / 60000
    if (minutes < 5) return 5000      // 5秒
    if (minutes < 30) return 30000    // 30秒
    if (minutes < 60) return 60000    // 1分钟
    if (minutes < 360) return 180000  // 3分钟
    return 300000                      // 5分钟
  }

  reset() { this.disconnectTime = 0 }
  recordDisconnect() { this.disconnectTime = Date.now() }
}
```

### 7.2 重连流程集成

在 MQTT 客户端的 `onClose` 中调用 `recordDisconnect`，然后使用 `setTimeout` 延迟重连。

## 8. 电源管理

### 8.1 网络状态监听 (`NetworkMonitor.ets`)

```typescript
import connection from '@ohos.net.connection'

export class NetworkMonitor {
  private callback?: () => void
  startListening(onNetworkAvailable: () => void) {
    this.callback = onNetworkAvailable
    connection.on('netAvailable', () => this.callback?.())
  }
  isNetworkAvailable(): boolean {
    return connection.getDefaultNetSync() !== null
  }
  stopListening() {
    connection.off('netAvailable')
    this.callback = undefined
  }
}
```

### 8.2 电池监控

```typescript
  onChange(cb: (level: number, charging: boolean) => void) {
    battery.on('batteryChange', (info) => cb(info.batterySOC, info.chargingStatus === 1))
```

### 9.1 设备控制 (`DeviceController.ets`)

```typescript
import deviceControl from '@ohos.enterprise.deviceControl'
import { BusinessError } from '@ohos.base'
import { Want } from '@ohos.app.ability.Want'

const adminWant: Want = { bundleName: 'com.siyu.mdm', abilityName: 'EntryAbility' }

export class DeviceController {
  lock(cmd: CommandMessage): CommandResult {
    try {
      deviceControl.lock(adminWant, cmd.getString('mainText'), cmd.getString('subText'))
      return CommandResult.success(cmd.commandId, cmd.action, '已锁定')
    } catch (e) { return CommandResult.failure(cmd.commandId, cmd.action, (e as BusinessError).message) }

    deviceControl.unlock(adminWant)
    return CommandResult.success(cmd.commandId, cmd.action, '已解锁')
  }

  wipe(cmd: CommandMessage): CommandResult {
    deviceControl.wipe(adminWant)
    return CommandResult.success(cmd.commandId, cmd.action, '已执行恢复出厂')
  }

  reboot(cmd: CommandMessage): CommandResult {
    deviceControl.rebootDevice(adminWant)
    return CommandResult.success(cmd.commandId, cmd.action, '已执行重启')
  }

  resetPassword(cmd: CommandMessage): CommandResult {
    deviceControl.resetPassword(adminWant, cmd.getString('password'))
    return CommandResult.success(cmd.commandId, cmd.action, '密码重置成功')
  }
}
```

## 12. 快速开始（鸿蒙）

### 步骤 1：创建鸿蒙项目
- DevEco Studio → 新建 → Application → Stage 模型  
- 包名：`com.siyu.mdm`  
- 项目名：`SiYuMDM`

### 步骤 2：配置权限与 MDM 能力

在 `entry/src/main/module.json5` 中：

```json
{
  "module": {
    "name": "entry",
    "type": "entry",
    "requestPermissions": [
      { "name": "ohos.permission.MANAGE_ENTERPRISE_DEVICE_ADMIN" },
      { "name": "ohos.permission.ENTERPRISE_MANAGE_SYSTEM" },
      { "name": "ohos.permission.INTERNET" }
    ],
    "extensionAbilities": [
      {
        "name": "EnterpriseAdminAbility",
        "type": "enterpriseAdmin",
        "srcEntry": "./ets/enterpriseadmin/EnterpriseAdminAbility.ets"
      }
    ]
  }
}
```

### 步骤 3：实现 EnterpriseAdminExtensionAbility

`ets/enterpriseadmin/EnterpriseAdminAbility.ets`:

```typescript
import EnterpriseAdminExtensionAbility from '@ohos.app.ability.EnterpriseAdminExtensionAbility'

export default class MyEnterpriseAdmin extends EnterpriseAdminExtensionAbility {
  onAdminEnabled() { console.info('MDM 管理员激活') }
  onAdminDisabled() { console.info('MDM 管理员禁用') }
}
```

### 步骤 4：安装 MQTT 库

在 `entry/oh-package.json5` 中添加：
```json
"dependencies": { "@ohos/mqtt": "5.0.0" }
```

### 步骤 5：复制本文档中的核心代码
- `MqttClient.ets`
- `CommandHandler.ets`
- `mdm/` 下各个管理类

### 步骤 6：激活设备管理员
用户首次打开 App 时，需要调用 `enterpriseAdminManager` 激活组件。示例：

```typescript
import enterpriseAdminManager from '@ohos.enterprise.adminManager'

const want: Want = { bundleName: 'com.siyu.mdm', abilityName: 'EnterpriseAdminAbility' }
enterpriseAdminManager.activateAdmin(want, (err) => {
  if (err) console.error('激活失败', err.message)
  else console.info('激活成功')
})
```

### 步骤 7：启动后台长连接服务
使用 `@ohos.ability.backgroundTaskManager` 申请长时任务，防止进程被系统清理。
